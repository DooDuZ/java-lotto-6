## Lotto

### 기능 구현

---
### I/O
- 콘솔을 통한 문자열 입력 기능
- 콘솔을 통한 메시지 출력 기능

### Validator
- 구입 금액 검증
  - 입력 값이 숫자인지 확인합니다.
  - 입력 값이 1000으로 나누어 떨어지는지 확인합니다.
  - 입력 값이 0보다 크고 구매 제한 금액보다 작은지 확인합니다.
- 당첨 번호 검증
  - 입력 값을 ,로 구분하여 분리합니다.
  - 구분된 값들에 중복이 있는지 확인합니다.
  - 숫자가 1~45 사이의 값인지 확인합니다.

### Lotto
- 당첨 확인

### Exception
- 에러 메시지 생성
